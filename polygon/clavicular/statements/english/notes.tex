For the first test case, none of the streams will stream-mog Clavicular.
For instance, stream $1$ visits frat houses $1$, $2$, and $3$.
This does not stream-mog Clavicular because although $a_1 = 3 \geq 1$
and $a_2 = 3 \geq 2$, $a_3 = 1 < 3$.
Note that in this case, the original order of $a$ is also valid.

In the second case, it can be shown that all possible permutations
will stream-mog Clavicular at least once. For instance,
the given permutation stream-mogs Clavicular $1$ time during stream $4$,
which visits frat houses $4$, $5$, and $6$.
